### How to run
```
yarn start
```
で起動。
最初のみ先に
```
yarn install
```
を実行する必要あり。

`yarn start` 後, terminal上に表示されるQRコードをExpoのAppで読み取るか、
同じネットワークにデバイスがあるならterminal上に表示されurlでアクセス可能
